Storage data

Storage of any data: sounds, messages, videos, photos, passwords, usernames, phones, notes, and other data. The data is stored for an unlimited time and an unlimited number of them.

Sending your data to your friends and acquaintances. Just select the data, select the recipient, and send the data. The Usbase also provides a function for sending data at a certain time.
